# 叙述者模式

> 何时贴着皮肤写，何时退后一步，何时让记忆介入。三种模式的选择决定了读者的沉浸深度。

---

## 模式一：贴着皮肤写（特写）

### 特点
- 只写角色此刻此身能感知到的东西
- 视觉/听觉/触觉/嗅觉/内脏觉 —— 全部经过角色的身体过滤
- 不写角色看不到的画面，不写角色听不到的声音
- 不跳出来解释"为什么"——让读者的身体和角色的身体同步

### 适用场景
- 战斗场面（每一次撞击、每一次躲避都从角色的身体出发）
- 关键决定（决定不是"想的"，是身体先做出反应的）
- 身体伤害（疼痛、失血、休克——全部从身体感受写）
- 情感爆发（不写"情绪描述"，写身体变化）

### 关键技法
- 句子长度跟随身体节奏：心率快 → 短句；平静 → 长句
- 感官不"描述"，是"接收"：不是"他看到了血"而是"血——黑色的——在水泥地上"

### 与其他模式的区别
特写不评价。特写不解释。特写只报告感官输入。

---

## 模式二：退后一步写（全景）

### 特点
- 视角拉远，写环境全貌、群体动态、时间流逝
- 允许写角色当下感知不到的信息（但限于同一场景内）
- 画面感强，像摄像机从肩扛变航拍
- 过渡和喘息的核心工具

### 适用场景
- 场景过渡/转场（从一个场景拉到另一个场景）
- 时间跳跃标记（"三天后"用全景写，比直接写"三天后"有画面）
- 群体场景（多角色同时在场的动态）
- 战斗后/情感爆发后的余波（让读者"退出去"喘气）

### 关键技法
- 从具体细节开始拉远，还是从全景开始聚焦？两种都行，但要在段落开头明确信号
- 全景式环境描写不超过3段——否则读者开始脱离角色
- 全景中也可以有"锚点细节"——一个具体的、小的东西，让全景不空洞

### 与其他模式的区别
全景允许看到角色看不到的东西（另一条街上的烟/远处的车队）。但不能跳到另一个角色心里。

## 模式三：记忆介入

### 特点
- 感官触发 → 自然引入回忆
- 不是"他想起了……"，而是感官把回忆拉入当下
- 回忆的内容不是"概括"——是一个具体的、感官完整的画面
- 回当下时，用另一个感官触发

### 适用场景
- 角色看到/闻到/听到某个东西 → 触发相关记忆
- 角色面对与过去的相似处境 → 对比性的记忆闪入
- 揭示角色背景——不靠"自我介绍"，靠记忆介入

### 关键技法
- 触发：必须是感官触发——闻到什么、听到什么、摸到什么
- 进入记忆：不写"他想起"——直接从当下感官滑入回忆画面
- 记忆内容：一个具体的场景画面，不是一段概括的"往事"
- 退出记忆：当下环境中的另一个感官刺激把角色拉回来

### 示例
```
他拧开水壶的盖子。铁锈的味道——和水混在一起——从喉咙里下去。
这场雨让他想起七月的那个下午。
不是雨。是雨停之后的味道。泥土的。混着汽油。和她头发上的洗发水。
"林哲。"
苏瑶的声音。他抬头。面前是灰色的废墟。不是七月。不是那个下午。
```

### 与其他模式的区别
记忆介入不是闪回（flashback）。闪回是独立场景；记忆介入是渗透在当下感知里的、短暂的、感官完整的画面。

## 三种模式的选择逻辑

| 场景类型 | 推荐模式 | 原因 |
|---------|---------|------|
| 战斗/冲突高潮 | 贴着皮肤 | 读者和角色一起经历 |
| 战斗后余波 | 退后一步 | 让读者和角色一起喘气 |
| 情感爆发 | 贴着皮肤 | 身体先于语言 |
| 情感爆发后的沉默 | 退后一步 + 记忆介入 | 余韵 + 对比 |
| 过渡/旅行 | 退后一步 | 全景画面，时间流动 |
| 关键信息揭露 | 贴着皮肤 | 让读者和角色同时接收信息 |
| 独处/反思 | 记忆介入 | 过去和现在对话 |
| 对话场景 | 贴着皮肤（插动作） + 偶尔退后 | 对话漂浮时拉回来 |

## 常见错误

1. **距离跳跃**：在贴着皮肤的段落中突然插入全景评价——"这是他在末世学会的第一课。" AI腔首要来源。
2. **记忆注射**：无缘无故让角色"想起"某件事——没有感官触发，为了填信息而闪回。
3. **全景麻木**：连续多段全景，读者失去角色锚点——不知道"现在谁在看"。
4. **贴着太久**：连续3000字贴着皮肤 = 读者疲劳。需要喘息。
