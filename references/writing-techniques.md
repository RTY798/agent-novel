## 1. 感官描写工艺

### 五感+内脏觉的切换频率
每 300-500 字至少切换一次感官主导。连续 800 字纯视觉描写 → 读者开始"看默片"。

### 感官与情绪的对应
| 情绪 | 主导感官 | 原因 |
|------|---------|------|
| 恐惧 | 触觉 + 内脏觉 | 心跳/冷汗/肌肉紧绷 — 恐惧是身体先知道的 |
| 紧张 | 听觉 | 人在紧张时对声音过度敏感 |
| 悲伤 | 触觉 + 视觉 | 重量感/温度感 + 慢镜头般的视觉 |
| 愤怒 | 内脏觉 + 触觉 | 热/胀/紧绷 |
| 温暖/安全 | 嗅觉 + 触觉 | 熟悉的气味 + 柔软的质感 |
| 回忆 | 嗅觉 + 听觉 | 气味是时间机器；声音是空间锚 |

### "贴着皮肤写"技法
用角色的感官过滤世界。不写角色看不到/听不到/摸不到的东西。

- 错误：房间里有一张旧沙发，上面搭着一条褪色的毯子。
- 正确：苏瑶的手背碰到粗糙的织物——毯子，已经磨得看不出原来的颜色。她收回手。毛料的刺痒留在指尖。

**原则**：先写感官接收，再写意义判断。读者和角色同时发现世界。

**强制规则：环境描写必须通过当前POV角色的感官过滤。**

反例："夕阳染红了半边天。"——谁看到的？没有具体的眼睛，没有具体的位置。
正例："他眯起眼——才发现天已经烧成了橘红色。"——明确是他，眯眼动作说明光很强，他刚才没注意到。

**自查**：读完每段环境描写后能回答：①谁在看/听/摸/闻？②在什么位置？③情感状态如何影响感知？三个问题不能全部回答→重写。

## 2. 对话工艺

### 角色声音差异化（5个维度）
1. **词汇量**：3000 词的人 vs 1000 词的人，说同一件事用不同的词
2. **句长**：短句角色（行动型）vs 长句角色（思考型）
3. **口头禅/习惯句式**：不是"俺""洒家"这种表面标签，而是句式偏好。一个人习惯用反问，一个人习惯把结论放前面
4. **沉默方式**：有的人沉默是愤怒，有的人沉默是同意，有的人沉默是在想词
5. **节奏**：快接话 vs 慢半拍，打断别人 vs 被打断不吭声

### 潜台词
角色说的 ≠ 角色想的 ≠ 角色想要的。

三层检查：
- 他说了什么？（字面）
- 他为什么说这个而不是别的？（选择）
- 他期待对方听完做什么？（意图）

**潜台词强制训练**：写完一节对话后，如果所有话都在推进情节——说明潜台词缺失。插入至少一句表面上"不相关"的话。目的是打破"对话=情节推进器"的惯性，让对话同时成为"人物揭示器"。详见 `golden-finger-techniques.md` → 手法4：对话潜台词 → 潜台词强制训练。

### 对话锚定（防"漂浮对话"）
每 3 句连续对话，插入一个物理动作或环境感知：

```
"你确定？"
"确定。"
"我不信。"
苏瑶把烟头按进桌面。木头烫出一小圈焦痕。
"信不信由你。"
```

**不是打断对话节奏**——锚定句本身就是节奏的一部分。焦痕的那一行，比"她生气地说"有力十倍。

### 沉默的使用
- 问题抛出后角色沉默 → 比回答更有信息量
- 沉默 + 一个微小的动作 → 情绪密度最高
- 两个角色之间的沉默长度差异 → 权力关系

## 3. 节奏控制

### 句子长度
- 长句（>30字）：沉浸感、描述、内心活动。让读者"沉下去"。
- 中句（15-25字）：叙述推进、常规描写。流水感。
- 短句（≤8字）：紧张、果断、冲击。让读者"加速"。

**变化规律**：紧张处句长逐渐缩短，释放处句长逐渐拉长。

### 段落节奏
- 动作/紧张场景：50-150字/段。读者快速下移。
- 反思/描述场景：150-250字/段。节奏放缓。
- 手机屏一行 ≈ 25字。150字 ≈ 6行。250字 ≈ 10行。

### 场景呼吸
紧张场景后必须有喘息空间——哪怕只是两行环境描写。

```
枪声停了。他的耳朵在响。
窗外是灰色的天。没有云。什么都没有。
```

这两行让读者的心率回来。不可跳过。

### "平头"避免
相邻段落长度不要都落在同一区间（如连续5段都是200-250字）。这不是语法错误，但会让读者潜意识里感到"单调"。

## 4. 叙述距离控制

### 三种距离

| 距离 | 标志 | 适用 | 感官特点 |
|------|------|------|---------|
| **特写**（贴着皮肤） | 只写此刻此身的感知 | 战斗/关键决定/身体伤害/情感爆发 | 触觉+内脏觉主导 |
| **中景**（正常距离） | 写角色看到/听到/做到的事 | 日常互动/对话/行动 | 视觉+听觉主导 |
| **全景**（退后一步） | 写环境全貌、时间流逝、群体 | 过渡/休整/时间跳跃 | 视觉主导+抽象 |

### 距离切换信号
- 空行 = 最强的距离切换信号
- 环境描写句 = 中等信号（"天色沉下来的时候"）
- 感官切换 = 弱信号（从视觉切换到触觉，暗示距离拉近）

### 常见错误
在特写中插入全景式评价：
- 错误：他的手在抖——这是一个人在面对巨大危险时最真实的反应。
- 正确：他的手在抖。从指尖到手腕。整只手。他把手塞进口袋。

**全景式评价是 AI 腔的头号来源。**

## 5. 场景结构

### 场景三要素
每个场景必须有：**目标 → 阻碍 → 变化**。

如果场景结束时什么都没变——这个场景可以不写。

### 场景节拍
一个场景 4-6 个节拍。每个节拍推进一个微变化（认知/情绪/关系/物理状态）。

| 节拍 | 功能 | 示例（对峙场景） |
|------|------|-----------------|
| 1 | 锚定（入场感官） | 苏瑶看到门缝下的影子 |
| 2 | 建立（双方位置/意图） | 她认出那双鞋——不是他们的人 |
| 3 | 推进（第一次试探） | "你是谁？"对方没有回答 |
| 4 | 转折（新信息/意外） | 影子移动了——不是一个人，是三个 |
| 5 | 高潮（冲突升级） | 她摸到腰间的撬棍。金属是凉的。 |
| 6 | 退场（最后一个画面） | 门缝下的影子消失了。地板上留下两道湿的脚印。 |

不是每个场景都需要6个节拍。4个也行。但少于3个节拍 = 场景不完全。

### 找核（v10.3升级）

每个场景写之前先找**核**：核 = 角色在这个场景内经历的**内心运动**。三种核型：

| 核型 | 标记 | 定义 | 示例 |
|------|------|------|------|
| **变化型** | 🔄 | 从X到Y（明确的变化） | 晚蓉从麻木到重新感觉自己是一个人 |
| **试图变化但失败型** | ❌🔄 | 试图从X到Y，卡在中间 | 宇轩在斜峪关想说出真相→张嘴→发不出声→合上嘴 |
| **深化型** | ⬇️ | 从X到更深的X（不是变化，是加深） | 长乐宫解散后，欧阳从紧绷到更紧绷——像一根弦已经拉到极限，再拉就断了 |

**不是核的**：
- "晚蓉洗热水澡"（这是事件，不是变化）
- "七人走进万象城"（这是地点切换，不是变化）

**感官载体规则**：默认1个具象物承载核。例外——两种感官产生对比/接力时，允许2个载体（如欧阳同时听到铁锹声和呼吸声，同一个听觉通道的张力承载核）。

**感情线附加检查**：如果这场涉及感情互动，写前确认——结束后心理距离近了还是远了？没变→缺核。

**验证**：场景结束后，角色的内心状态和进入场景之前相比，发生了什么变化（或试图发生什么变化/或加深了什么）？答不上来→这个场景没找到核。

详见 `references/nucleus-first-method.md`。

### 入场和出场
- **入场**：场景的第一个感官印象。让读者立刻站在角色的肩膀上。
- **出场**：场景的最后一个画面。这是天然的"微钩子"——让读者想看到下一个场景。

### 删减法
写完逐段问："删掉这一段，会影响读者理解接下来发生的事情吗？"
不会 → 删。
会，但可以用半句话替代 → 删掉，补半句话在前面或后面。

## 6. 情感传达技法

### 核心原则：不写情绪词，写身体

| 不要写 | 写这个 |
|--------|--------|
| 他害怕 | 他的手在抖。从指尖到手腕。他把手塞进口袋。 |
| 她伤心 | 她把没喝完的水倒掉。盯着空杯子。看了很久。 |
| 他愤怒 | 他的后槽牙咬紧了。又松开。又咬紧。 |
| 她紧张 | 她的呼吸从鼻子里出来——两次——然后停了。 |
| 他犹豫 | 他的脚抬起来。放下去。抬起来。换了个方向。 |

### 情感升级梯度
1. **身体反应**（不自觉的）：手抖/呼吸变快/出汗/肌肉紧绷
2. **行为变化**（半自觉的）：压低声音/避开对视/加快脚步
3. **决定**（自觉的）：选择做一件之前不会做的事
4. **对他人产生影响**：该决定改变了另一个人

**规则**：每个梯度至少停留一个节拍。从"身体反应"直接跳到"改变世界" = 假高潮。

### 反向技法
- 极度愤怒 → 写冷静。角色越愤怒，动作越慢。说话越少。
- 极度悲伤 → 写麻木。角色不哭。盯着一个东西看。忘了时间。
- 极度恐惧 → 写听觉过敏。每一声碎玻璃都像锤子敲头骨。

## 7. 描写工艺

### 具体 > 抽象
- "一棵树" → "一棵被雷劈掉一半的老槐树"
- "房间里有一张桌子" → "桌子腿下面垫着一本卷边的电话簿"
- "他穿着旧衣服" → "他的袖口磨出了线头。右肩有一块褪色的印子——常年背单肩包的位置。"

### 动态 > 静态
给静态画面一个"时间的痕迹"：
- "墙上有一面镜子" → "墙上的镜子裂了一条缝，从左上角到右下角——不是今天裂的，灰尘已经填进了缝里。"

### 细节的选择标准
只选满足以下至少一个条件的细节：
1. **揭示角色**：这件东西/这个习惯/这个痕迹 — 能让你推断出这个人的什么？
2. **推进情绪**：这个细节让读者感到什么？
3. **暗示历史**：这个细节暗示了之前发生过什么？

其余细节 —— 删。

### 描写的节奏功能
描写可以用来"刹车"：
- 连续三页快节奏动作后 → 一段长描写让读者喘气
- 情感爆发后 → 环境描写让余韵散开

描写也可以用来"蓄力"：
- 大战前的安静 → 细节描写积累紧张
- 重要对话前 → 环境描写建立氛围
