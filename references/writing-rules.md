# 类型与规则调整建议

> **原则**：6条硬性规则是基线。不同类型的叙事需求不同，以下建议供参考。
> **最终裁决权在作者。** 如果你觉得放宽某条规则让文字更好，无视建议。
> **不自动覆写参数。** 这里提供的是建议和原理，不是新的规则表。

---

## 规则1：抽象情绪词（基线：0容忍）

禁止词：感到、觉得、知道、认为、明白 — 禁止出现在叙述中。

### 各类型调整建议

- **言情/情感向**：可适当放宽。情感描写是核心，但注意——避免"她感到一阵温暖"这种模板化表达。放宽不等于放弃。建议：保留 QC 检测，阈值从 0 改为 ≤3 次/千字，由作者根据实际文字效果调整。

- **悬疑/惊悚**：保持 0 容忍。氛围靠感官营造——"走廊里有声音"比"他感到走廊里有危险"恐怖十倍。

- **奇幻/科幻/末世**：保持 0 容忍。行动驱动型叙事，抽象情绪词会直接削弱沉浸感。读者是通过角色的眼睛看世界的——角色的眼睛不会看到"一阵恐惧"。

- **轻小说/第一人称**：视叙事语气而定。如果叙事者本身就是话多的吐槽型角色，适当使用"觉得""感到"可以塑造声音。但依然要警惕——即使是第一人称，"我感到害怕"也不如"我后背的汗把T恤粘住了"有力。

### 原理
抽象情绪词 = 作者替读者做了感受工作。读者不需要感受，只需要被"告知"。这违背了"贴着皮肤写"的核心原则。

## 规则2："不是X是Y"句式（基线：≤2次/章）

### 各类型调整建议

- **言情**：可放宽至 ≤3 次。情感对比（"不是不爱，是不知道怎么爱"）是言情类型的常见修辞。但注意：一章内超过3次仍然会产生"作者在反复解释"的感觉。

- **悬疑**：建议收紧至 ≤1 次。精简句式增强紧张感。对比句式自带"停顿解释"的节奏，会打断悬疑的连贯性。

- **其余类型**：保持基线 ≤2 次。偶用一次有冲击力（"不是丧尸。是人。"），用多了是作者在代替读者思考。

### 原理
"不是X是Y"本质上是一种**纠正性解释**——作者跳出来说"不是你想的那样，其实是这样的"。偶尔用有张力。连续用 = 作者不信任读者的理解力。

## 规则3：同一角色连续说话 ≤ N 句（基线：3句）

中间必须插入动作/环境/沉默。超过3句开始"漂浮"——读者看到的是台词，看不到说话的人。

### 各类型调整建议

- **言情**：可放宽至 4 句。对话是言情核心，情感交流需要一定的对话深度。但依然要坚持"锚定"原则——每 3-4 句插入物理动作或环境感知。

- **悬疑/惊悚**：建议收紧至 2 句。紧张感需要更频繁的物理锚定——角色的微表情、环境的变化、声音的细节。

- **轻小说/对话驱动型**：可放宽至 5 句。但要确保角色声音足够差异化，读者即便看不到锚定也能分辨谁在说话。

- **其余类型**：保持基线 3 句。

### 原理
对话超过3句开始失去物理空间感。读者脑中只剩两个漂浮的声音。锚定句把声音拉回身体。

## 规则4：段落 ≤ N 字（基线：250字）

250 字 ≈ 手机屏 10 行。超过这个长度，读者开始跳读。这是生理事实，不是文学判断。

### 各类型调整建议

- **言情/情感向**：保持 250 字。对话多天然控制段落长度。

- **悬疑/惊悚**：建议收紧至 200 字。短段落制造呼吸急促的阅读节奏。手机屏快速下滑的感觉本身就是紧张感的一部分。

- **奇幻/科幻**：保持 250 字。世界观描述可能需要较长的段落，但要警惕——描述一段 300 字的魔法体系，不如三段 100 字的发现过程。

- **轻小说**：保持 250 字。但如果发现段落经常接近上限，考虑拆段。

### 原理
不是"长段落不好"。是手机阅读时代读者的注意力窗口就是 250 字左右。超过这个长度，读者会潜意识地开始"找下一个自然分段点"而非"理解内容"。

---

## 规则5：同一句式不连续复用（基线：不变）

开头/结尾/段落结构不连续复用。这条规则与类型无关，保持基线。

### 原理
"他做了A。他做了B。他做了C。"这不是风格，是模板 bug。句式变化本身就是叙事节奏的一部分。

---

## 规则6：不写角色解释他人心理状态（基线：0容忍）

### 各类型调整建议

- **言情**：**适度放宽**。言情叙事的核心卖点之一就是角色的内心世界。但要区分——
  - 允许：POV 角色对自己的情感进行觉察和描述（"她发现自己一直在等他开口"）
  - 不允许：POV 角色解读对方心理（"她知道他在撒谎"→ 改为"他说话的时候没有看她。"）
  - 折中：用观察和猜测替代确定——"他的眼睛往左边瞟了一下。也许是在想词。也许是在撒谎。"

- **悬疑/惊悚**：保持 0 容忍。不确定性是悬疑的核心资产。角色"知道"别人在想什么 = 提前泄题。

- **其余类型**：保持 0 容忍。

### 原理
没有人能读心。角色当然也不能。如果角色需要猜测——让他猜测，但用"也许""可能""如果"标记。不要让他"知道"。

---

## 各类型字数范围建议

| 类型 | 建议字数/章 | 原因 |
|------|-----------|------|
| 末世/公路（基线） | 1800-5000 | 动作+过渡交替 |
| 言情 | 2000-6000 | 情感需要更多铺垫和内心活动 |
| 悬疑 | 1500-4000 | 短章保持节奏，长章容易泄劲 |
| 奇幻/科幻 | 2000-5000 | 世界观描述需要空间 |
| 古风 | 1800-5000 | 与基线一致 |
| 轻小说 | 1500-3500 | 短章快节奏，符合读者预期 |

> 这些范围是建议，不是限制。如果故事需要 8000 字一章，写 8000 字。
