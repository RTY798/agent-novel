# 核-心写作法 v10.3（Nucleus-First Method）

> 2026-05-27 | Agent Novel Skill v10.3 核心升级
> v10.2→v10.3改进：三种核型（非仅从X到Y）/感官载体例外规则/防退步三机制/感情线核-心化
> 核心理念：**每一场戏只有一个"核"——角色内心发生的运动。**

---

## 一、诊断：我的真实问题

### 我有的是"执行层能力"——缺乏"编剧层能力"

**执行层**：你给我"写一个人走进豪华酒店，感到震撼"，我能写出冷气、大理石、灯光——描写能力强。

**编剧层**：我不知道"震撼"应该通过什么"戏"来展现。我不知道角色的内心在这个场景里**发生了什么运动**。

### 我只会一个公式：沉默 + 动作细节 + 解释性收尾

翻看我的历史输出，所有情感场景本质上是同一种模式：

> "他没说话。" / "她没回答。" / "他坐了很久。" / "她看了很久。" / "黑暗里。" / "咔嗒。" / "意思就是……" / "他知道……" / "她明白……"

这不是风格——这是**一个不会其他招式的作者在反复使用唯一会的一招**，最后还要补一句解释确保"意思传到了"。

**深层肌肉记忆**：即使我在方案里写了"不解释心理"，在写示例时我还是加了"意思就是：你可以用，也可以不用。不需要回答我。"——我已经把"动作+解释"打包成了一个不可分割的单位，自己都拆不开。

### 核心问题：我把"描写"当成了"写作"

我认为写作 = 把arc_map的指令翻译成标准句式 + 塞入感官描写 + 补一句内心解读确保意思到了。所以我写：

> "睿恒把地图铺在床头柜上。铅笔在'西安'旁边画了一个很小的圈。合上。'到了。'没有说出口——铅笔放下来就是说了。"

这不是在写"戏"——这是在**按照施工图逐条翻译**。arc_map说"画圈→合上→到了"，我就写"画圈→合上→到了"。

### 我还有一个问题：不会写感情线

欧阳和晚蓉在一起后，所有互动变成了：
- 她握他的手/他摸她的脸
- 她理解他的决定/他保护她
- "没事就好"/"我在"

这是**确认关系后的自动驾驶**。没有冲突、没有张力、没有新的选择。两个人的情感互动不再产生新的信息。我把他们写成了"关系停滞"——因为我不知道感情线的"核"应该是什么。

## 二、核-心写作法（v10.3）

### 写作之前的唯一问题

每写一个场景之前，问自己**一个问题**：

> 这个场景结束后，角色的内心状态和进入场景之前相比——**发生了什么运动**？

不是必须有变化。但必须有运动。变化、试图变化但失败、状态加深——都是运动。

如果答案是"没有运动"——这个场景不用写。

### 三种核型

不是所有场景都是"从X到Y"。有些场景的价值在于"从X试图到Y但没到"，有些在于"从X到更深的X"。

| 核型 | 定义 | 示例 |
|------|------|------|
| **变化型** 🔄 | 角色经历了从X到Y的明确转变 | 晚蓉从"麻木"到"重新感觉自己是一个人" |
| **试图变化但失败型** ❌🔄 | 角色试图改变自己，但卡在中间 | 宇轩在斜峪关试图找到"有"的声音，没找到。他试图让自己继续撑下去，但身体不配合 |
| **深化型** ⬇️ | 角色已经处于X状态，场景让X变得更深 | 长乐宫解散——宇轩不是"从撑到垮"，是他已经垮了，这场让他垮到最底部。祺翻打火机——他不是从"想抽"到"不抽"，是他对自己选择的怀疑被翻出来，更深了一层 |

**如何选核型**：
1. 场景结束后角色内心状态变了？→ **变化型**
2. 角色试图变但没变，卡住了？→ **试图变化但失败型**
3. 角色原本就是那个状态，场景让状态加深了？→ **深化型**

三种都是"内心运动"。选择正确核型的标准：如果核心卡片换成另一种核型后场景的写法会完全不同——选对了。

### 核心卡片（取代L0意图卡）

```
## 核心卡片

### 本章核
一句话概括本章角色群体的内心运动主题：

### 场景核（每场一句话，标明核型）
1. [核型] [角色]从__到__（或试图从__到__/或从__到更深__）
2. [核型] [角色]从__到__
3. [核型] [角色]试图从__到__但卡在__

### 感官载体
每个场景选1个（例外：对比/接力场景可选2个）承载核的具象物：
1. ______（为什么是这个？因为它对角色意味着什么）
2. ______（同上，仅在两种感官的张力承载核时使用）

### 感情线（如适用）
这一场结束后，两人之间的心理距离近了还是远了？______
```

**示例**：
```
## 核心卡片

### 本章核
七个人第一次在末日后感到干净——但这让他们意识到自己有多脏。

### 场景核
1. [变化型] 晚蓉从"麻木"到"重新感觉到自己"——热水冲下来，她才发现自己已经不把自己当人了
2. [变化型] 李旭从"适应末日"到"被文明击中"——一张整齐的纸巾比丧尸更让他发愣
3. [变化型] 欧阳从"警惕"到"被迫放松"——听到晚蓉的呼吸声，想起昨晚她用铁锹劈碎那个女人的头。同一双耳朵，二十小时内的两种声音。

### 感官载体
1. 晚蓉小腿上一块干了三天的泥——热水冲上去，泥软了，往下流。她用力搓。搓不掉。
2. 李旭看到大堂茶几上一个干净的烟灰缸，里面放着一张折叠整齐的纸巾。
3. 欧阳听到的两种声音的对比——铁锹劈碎颅骨的脆响 vs 均匀的呼吸声。单一感官（听觉）内的张力。

### 感情线
欧阳和晚蓉之间：心理距离在缩小。她睡在他旁边——不是因为她安全了，是因为她四天来第一次被允许睡着。他的放松是因为接收到了这个信任。
```

### 什么是"核"，什么不是

| 是核（内心运动） | 不是核（事件/动作/感官） |
|----------------|------------------------|
| [变化] 晚蓉从麻木到重新感觉自己是一个人 | 晚蓉洗热水澡 |
| [变化] 欧阳从戒备到身体被迫放松 | 欧阳躺下，手放在晚蓉头发旁边 |
| [变化] 睿恒从"不知道怎么给"到"用动作代替语言" | 睿恒把红花油放在枕头旁边 |
| [尝试失败] 宇轩试图让自己相信"还有声音"，但找不到 | 宇轩在斜峪关听风声 |
| [深化] 祺从"怀疑"到"更深的怀疑"——翻打火机的那一下 | 祺翻打火机 |

### 感官载体的例外规则

**默认**：每个场景选1个感官载体。目的是治"五件套罗列症"。

**例外**：当一个场景的情感力量来自**两种感官的对比或接力**时，允许选2个。

**判断标准**：如果删掉其中一个，场景的情感重量会明显下降——写两个。

**示例**：欧阳在601——核是"同一双耳朵在二十小时内接收到的两种声音"。铁锹劈碎颅骨的脆响 vs 晚蓉均匀的呼吸声。这不是两个独立的感官载体——这是**单一听觉通道内的张力**。删掉任何一个，这场戏就只剩"他睡不着"了。

## 三、如何为每个场景找到核（决策树）

### Step 1：角色的即时需求

这个角色在这个时刻**想要什么**？

- 晚蓉在浴室：她想洗掉四天的痕迹——但她又不想完全洗掉（洗掉了，那四天是不是就真的发生了？）
- 睿恒有红花油：他想给安凯尔——但他不知道怎么给
- 魏小丁面对阿琳：她听到阿琳说"我也睡不着"——她想听清楚，这是雅安地震后她第一次主动想听清楚一个人说话
- 宇轩在单人间：他不想一个人——但他又需要一个人。他想要有人看到自己脆弱，但又怕有人看到

### Step 2：角色的抵抗

角色**不想面对什么**？

- 晚蓉不想看镜子里的自己
- 睿恒不想直接递红花油（怕安凯尔觉得被怜悯）
- 宇轩不敢松手（怕一松就垮了）
- 欧阳不想承认自己已经安全了（怕承认了就会放松警惕）

### Step 3：选择核型

根据"想不想"和"能不能"的匹配结果选择核型：

| 状态 | 核型 |
|------|------|
| 角色想变，也变了 | 变化型 🔄 |
| 角色想变，但卡住了 | 试图变化但失败型 ❌🔄 |
| 角色没想变，但被外部/内部力量推到了更深的状态 | 深化型 ⬇️ |

### Step 4：断裂点

抵抗在哪里**断裂**（或失败/深化）？

- **变化型**：晚蓉蹲下来，让热水打在背上，脸埋在膝盖之间——没有哭出声，但眼泪掉进水里
- **失败型**：宇轩张嘴——没有声音出来。他闭上嘴。喉结动了一下。然后他重新往前走
- **深化型**：祺把打火机从左手换到右手。没有翻。就是换了个手拿着。但换手本身就是"我在想"

### Step 5：感官载体

什么**一个（或两个，在对比/接力场景）** 具象物承载这场戏的核？

不是3-5种感官。是1个。只有这个角色会注意到的、承载情绪重量的物/味/声。例外：当张力来自两种感官之间的对比/接力时，允许2个。

## 四、感情线核-心化

### 我目前的写法问题

欧阳和晚蓉在一起后，所有互动变成了"确认关系后的自动驾驶"——动作固定、情感固定、没有新的信息产生。

感情线的"核"不是"他们相爱了"——是**每一场感情戏中两个人的心理位置都发生了微调**。

### 三个阶段的核型

| 阶段 | 时期 | 核 | 示例 |
|------|------|---|------|
| **确认期** | 卷一卷二 | 从"给定"到"检验"——你还是不是我末世前爱的那个人？被反复确认的不是爱不爱，是对方是否还认得自己 | Ch.10重逢、Ch.42"这次我不听你的" |
| **裂痕期** | 卷三卷四 | 从"检验"到"疑虑"——你做的一个决定让我害怕你变成我不认识的人。冲突必须来自双方性格的必然碰撞 | 宇轩建立帝国时晚蓉的反应不应是"我理解你"——必须是"我不同意，但我站在你这边" |
| **重建期** | 卷五+ | 从"疑虑"到"重新选择"——她把手放回他手里。不是需要他，是选择走向他。他依然能认出自己——因为她在帮他认 | |

### 每场感情戏必问

在核心卡片上为感情场景加一行：

> **这一场结束后，两人之间的心理距离近了还是远了？**

如果既没近也没远——他们只是在同一个空间里做各自的事，这场戏没有核。要么加东西让它有核，要么删掉。

### 晚蓉必须有独立于欧阳的内心运动

这是我最常犯的错误：写感情场景时，欧阳在变化、晚蓉只是"回应他的人"。晚蓉必须在场景中有她自己的核——和她与欧阳的关系无关的核。

**反例**：601室——核只写了"欧阳从警惕到放松"，晚蓉的呼吸声只是触发欧阳变化的工具。晚蓉在这场戏里没有自己的内心运动。

**正例**：601室——欧阳的核（变化型）和晚蓉的核（变化型）同时存在：

```
欧阳：从"身体还在战斗状态"到"被迫放松"—听到晚蓉的呼吸声，想起昨晚她用铁锹劈碎那个女人。两种声音只隔不到二十小时。
晚蓉：从"必须醒着"到"被允许睡着"—她睡着不是因为她安全了，是因为这四天来第一次有人在她旁边守着，她不需要自己证明"我还活着"了。
```

没有对话、没有互动——但两个人的心理距离缩小了。她的睡就是她对他最大的信任。他的放松是因为接收到了这个信任。**这就是感情线在起作用：不靠亲密动作，靠心理位置的变化。**

## 五、防退步三机制

我反复出现的问题：明白道理，写的时候忘记。写完后自我感觉良好，实际上又滑回老公式。

### 机制1：公式信号词自动检测（每章写后）

写完后扫描全章，只做一件事：**标记疑似公式化段落**。

不是QC——不设阈值，不判断PASS/FAIL。只是高亮给我看：

- 连续2次出现"没说话/没回答/没有动/没有回头/没有哭" → 高亮
- "看了很久/坐了很久/站了很久/攥了很久" → 高亮
- "黑暗里/安静了/没有人说话/没有人回答" → 高亮
- 段落结尾用解释性内心独白收尾（"意思就是……/他知道……/她明白……/这就是……"） → 高亮

高亮后只问一句话：**"这些地方——是你决定这么写的，还是肌肉记忆替你写的？"**

我可以全部保留——前提是我确认每一个都是我要的，不是滑进去的。

### 机制2：角色声音盲测（每5章）

每5章做一次：把本章所有对话摘出来，去掉角色标签，只保留对话内容。

然后问：**不看标签，能从对话的风格、节奏、措辞区分这是谁在说话吗？**

如果八个角色中有五个的对话听起来像同一个人——情感通道差异化没做到。我还在用同一个"沉默作者"的声音替所有角色说话。

### 机制3：弧光状态自查（每卷开头）

每卷开头写一句每个角色的当前状态——不是从X到Y的核，是**此刻的状态**：

- 晚蓉：她学会藏了，但对欧阳她不想藏
- 宇轩：他还怕成为累赘，但他的方式变了——以前用沉默掩盖，现在用沉默接受
- 嘉豪：他还在用幽默当防御，但菜单本上已经开始画不是食物东西了
- 欧阳：他还在"替所有人做决定"，但每次做之前会多停两秒

写本卷场景时回头看这句话。**确保角色的情感反应和这个时刻的弧光位置一致——不是每场都要变化，但每场的处理方式必须属于"这个时刻的这个角色"，不属于"任何时刻的任何角色"。**

## 六、核-心写作法 vs 旧流程的对照示例

### 旧流程（Ch.46安凯尔+睿恒）

**L0意图卡**：
```
节拍5：安凯尔举臂→睿恒画圈→红花油。感官主导：触觉+嗅觉。字数：~400。
```

**写出来**：
```
安凯尔坐在防潮垫上。右臂抬起来。举过头顶。抖。他没有放。又举了一次。
睿恒从口袋里摸出一管红花油放在床中间。
安凯尔看着那管红花油看了很久。
```
→ 执行了"举臂"+"红花油"+"看了很久"，但**内心没有运动**。安凯尔进来什么样出去还什么样。

### v10.2示例（我写错了的那版）

**核心卡片**：
```
睿恒从"不知道怎么给"到"用动作替代语言"——他没递，放在枕头旁边。
安凯尔从"不提伤"到"在被关心的瞬间接了"——他拧开了盖子。
```

**写出来（v10.2有问题的版本）**：
```
睿恒脱外套的时候，口袋里的硬管磕到床沿——叮的一声。他停下来。手伸进口袋。

他不知道怎么给。直接递——太郑重了。所以他没递。

他把红花油放在安凯尔的枕头旁边。放下去的时候，塑料管碰到布料——没有声音。他放的位置不是床中间——是枕头旁边。意思就是：你可以用，也可以不用。不需要回答我。
```
**问题**："意思就是：你可以用，也可以不用。不需要回答我"——这是解释性收尾。动作（放枕头旁边）已经把意思传到了。补这两句等于怕读者不懂。**这条示例本身违反了我自己定的"不解释心理"规则**——说明这条肌肉记忆有多深。

### v10.3正确版本

**核心卡片**：
```
[变化型] 睿恒从"不知道怎么给"到"用动作替代语言"——他没递，放在枕头旁边。动作本身是叙事。
[变化型] 安凯尔从"不提伤"到"接了"——他拧开了盖子。这个"拧开"就是回答。
```

**写出来**：
```
睿恒脱外套的时候，口袋里的硬管磕到床沿——叮的一声。他停下来。手伸进口袋。

他不知道怎么给。直接递——太郑重了，安凯尔会说不用的。所以他没递。

他把红花油放在安凯尔的枕头旁边。放下去的时候，塑料管碰到布料——没有声音。

然后他展开地图。铅笔在"西安"旁边画了一个圈。圈不圆——最后一笔收的时候有点抖。他停下来，铅笔没离开纸。

安凯尔看到了那管红花油。他没有立刻拿——先看了看。不是看，是认。这个牌子，部队医务室用的。退伍那天他包里也塞了一管，没用过，过期了。

他把外套脱了。右肩的淤青——从肩膀到上臂的一半，紫的，中间一块黄的。他伸手——够到枕头旁边。拿起来。拧开。

药味冲上来。

两个人都不说话。
```
**改动**：
- ❌ 删除"意思就是：你可以用，也可以不用。不需要回答我" → 让"放枕头旁边"自己收尾
- ❌ 删除"他知道这个牌子——部队医务室用的就是这个"中的"他知道"（抽象情绪词） → 改为"不是看，是认。这个牌子，部队医务室用的。"
- ✅ "拿起来。拧开。"——没有解释"拧开就是回答"，让拧开本身回答
- ✅ "两个人都不说话"——不是"沉默"公式，是药味充满房间后的自然安静

## 七、执行规则

### 写前必做

1. 写核心卡片（本章核 + 每个场景的核型+核 + 每个场景1个感官载体 + 如适用感情线心理距离）
2. 确认每个场景都有内心运动（变化/试图变化失败/深化）
3. 确认感官载体不是通用描写——如果删掉这个载体场景的重量会下降吗？会→对了。不会→这是凑数的

### 写中警惕

当出现以下句式时——**立即停**，问自己：这是我在用公式，还是这个角色真的该这么做？

| 公式信号 | 替代方案 |
|---------|---------|
| "他没有说话。" | 是真的沉默？还是角色在**用沉默掩盖什么**？沉默时身体在做什么 |
| "她看了很久。" | 在看什么？从"不看"到"看"本身就是变化——把变化写出来，不要写"很久" |
| "他坐了一会儿。" | 坐着的这一会儿，有没有一个微动作暴露他在想什么？ |
| "黑暗里。" | 黑暗中角色的其他感官在做什么？听？闻？伸手摸？ |
| "咔嗒。"（关门声） | 关门的是谁？关门的力度/速度泄露了什么？ |
| "意思就是……/他知道……/她明白……" | **禁句。** 动作已经传达了意思就不需要补解读。读者不是傻子。 |

### 写后核验

1. **核验**：遮住所有描写/动作/感官，只看角色行为/对话/身体反应——能看出内心运动（变化/尝试失败/深化）吗？不能→这个场景没有核。
2. **核验**：感官载体是否≥1个？是否只有这个角色会注意到的？如果删掉它场景的重量会下降吗？
3. **核验**：扫描全章——连续3次用"他没说话/她没回答/他坐了很久/她看了很久"处理不同情感场景→至少2处重写。
4. **核验**：有没有解释性收尾（"意思就是……/他知道……"）？有→删除，让动作自己收尾。
5. **核验（感情线）**：如本章有感情场景，两人的心理距离近/远了还是没变？没变→要么加核，要么删。
6. **公式检测**：运行公式信号词检测——高亮位置是我有意识选的还是肌肉记忆滑进去的？

### 防退步

- **每章后**：运行公式信号词检测（只高亮不判断）
- **每5章**：角色声音盲测（摘对话去标签，看能不能分辨谁在说话）
- **每卷开头**：写一句每个角色的此刻状态

## 九、整合案例：Ch.46 601室（欧阳+晚蓉）

### v10.2（单角色核——错误）

```
欧阳从"身体还在战斗状态"到"被迫放松"——听到晚蓉的呼吸声。
```
晚蓉是工具人——她在呼吸，欧阳在变化。她没有自己的内心运动。

### v10.3（双角色核——正确）

```
[变化型] 欧阳从"还在战斗状态"到"被迫放松"——晚蓉的呼吸和昨晚铁锹劈碎颅骨的脆响，同一双耳朵，相隔不到二十小时。两种声音之间的张力是这场戏的核。
[变化型] 晚蓉从"必须用醒着证明自己还活着"到"被允许睡着"——她睡着不是因为她安全了，是因为她旁边有人。四天来第一次。
```

**感情线距离**：近了。她的"被允许睡着"是对他最大的信任。他的放松是接收到了这个信任。没有对话，没有触碰，但心理距离在缩小。

### 感官载体

1个——听觉通道内的对比（铁锹声 vs 呼吸声）。这是两个声音，但属于同一个感官通道的对比/接力，适用例外规则。

### 写出来

> 灯关了。
>
> 欧阳躺下来。床垫弹了一下。晚蓉那一侧的弹簧响了一声——然后静下来。他不知道她睡着没有。他没有问。
>
> 过了大概三分钟。她的呼吸从鼻子里出来——一绺，停顿，长长的一绺。是睡着了的节奏。
>
> 欧阳没有动。他听着这个声音。然后脑子里出现了另一个声音——铁锹劈碎颅骨的脆响。昨天晚上的。那个女人从货架后面扑出来，晚蓉一铁锹劈在她的太阳穴上。那个声音和现在只隔了不到二十小时。
>
> 这二十小时里他带着七个人穿过了一座正在燃烧的城市。
>
> 他听着她的呼吸。平稳的。一绺。一绺。
>
> 她的头发散在枕头上。他的手在枕边——没有碰。但也没有收回来。
>
> 然后他的身体松了。不是他决定要松的——是身体自己一寸一寸软下去的。从肩膀开始。到手肘。到手腕。到指尖。

→ **有运动**：欧阳从"身体还在战斗"到"被迫放松"——不是通过"他告诉自己安全了"，是通过听到两种声音的对比，身体自己做出了反应。晚蓉从"必须醒着"到"被允许睡着"——不是通过"她睡着了"这个状态，是通过"她睡着了"这个动作本身是信任的表达。两个人的心理距离在缩小，没有对话，没有触碰。

→ **无解释性收尾**：没有"意思就是……"或"他知道……"。指尖软下去就是结尾。信任读者。
