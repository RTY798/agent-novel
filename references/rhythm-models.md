# 通用场景节奏模型

> 两个模型覆盖 80% 的场景节奏需求。选择模型 → 按节拍分配感官和情绪 → 写作。
> 不是模板——是方向。具体实现由故事决定。

---

## 模型A：上升型（冲突推进）

### 适用场景
战斗、对抗、揭露真相、追逐、紧张对话、高压决定

### 核心参数

| 参数 | 值 |
|------|-----|
| 节拍数 | 5-6 |
| 情绪曲线 | 持续上升（允许1次小回落再冲高——"几乎得救 → 再次陷入危机"的锯齿上升） |
| 叙述距离变化 | 全景（锚定战场）→ 中景（推进）→ 特写（高潮） |
| 感官侧重 | 视觉+听觉为主（建立空间和威胁方向），高潮节拍插入触觉/内脏觉（身体受伤/体能极限） |
| 段落节奏 | 前段正常（100-200字），高潮段逐步缩短（50-100字），结尾段最短（≤50字强调冲击力） |

### 节拍分解

```
节拍1：锚定
  功能：建立场景空间、威胁方向、角色位置
  叙述距离：全景或中景
  感官主导：视觉（空间布局）+ 听觉（远处的声音/靠近的声音）
  情绪：低

节拍2：接触
  功能：威胁进入可感知范围。第一次试探或观察。
  叙述距离：中景
  感官主导：视觉+听觉
  情绪：开始上升

节拍3：升级
  功能：威胁不再是远处的东西——直接作用到角色身上
  叙述距离：中景→开始推近
  感官主导：视觉+听觉+触觉开始介入
  情绪：快速上升

节拍4：危机（可选·回升前的低点）
  功能：角色几乎失败/失去某个优势。最长的一瞬间。
  叙述距离：特写
  感官主导：触觉+内脏觉（疼痛/冲击/体能极限）
  情绪：冲高后急降 → 再冲高（如果是最矮的谷）

节拍5：高潮
  功能：决定性的一击/选择/结果
  叙述距离：特写
  感官主导：触觉+内脏觉（身体先于意识）
  情绪：最高点

节拍6：余波（退场画面）
  功能：战斗结束后的第一个画面
  叙述距离：中景或全景
  感官主导：听觉（安静/耳鸣/远处的声音）+ 视觉（静止的画面）
  情绪：快速回落 → 留下一个情绪锚点
```

### 核心技法
- 节拍之间不允许"缓和"——上升型的每一个节拍都比前一个更紧张
- 节拍4（危机）如果不需要可以跳过，直接从升级到高潮
- 结尾的余波节拍是天然钩子位——可以在余波中砸信息炸弹或做反差沉默

## 模型B：下降型（情感/反思）

### 适用场景
情感爆发后的余波、角色独处、旅途过渡、安静的告别、伤口的"第一道愈合"

### 核心参数

| 参数 | 值 |
|------|-----|
| 节拍数 | 4-5 |
| 情绪曲线 | 下降或U型（低沉 → 微光 → 再回落 → 回升/接受） |
| 叙述距离变化 | 特写（从角色的身体出发）→ 中景 → 全景（逐渐拉远，让读者"退出去"） |
| 感官侧重 | 触觉+内脏觉为主（沉重/冷/空/酸），辅以视觉（慢镜头般的画面） |
| 段落节奏 | 允许较长的段落（150-250字），节奏放缓。偶尔一个超短段制造节奏变化。 |

### 节拍分解

```
节拍1：锚定（身体状态）
  功能：从角色的身体感受开始——累/疼/冷/空
  叙述距离：特写
  感官主导：触觉+内脏觉
  情绪：低点

节拍2：环境渗透
  功能：角色开始注意到周围的环境——透过情绪滤镜看到的世界
  叙述距离：特写→中景
  感官主导：视觉（慢镜头般）+ 听觉（安静/远处的声音）
  情绪：持续低点，开始有微小的变化

节拍3：微光（可选·U型谷底）
  功能：一个小的、温暖的东西进入感知——远处的灯光/风声里的一个旧旋律/一个不起眼的善意
  叙述距离：中景
  感官主导：视觉+听觉（那一刻的注意）
  情绪：小幅回升

节拍4：拉远（退场）
  功能：画面逐渐拉远。角色变小。环境变大。时间感恢复。
  叙述距离：中景→全景
  感官主导：视觉（航拍感）
  情绪：回升到中性偏暖（不是"开心"——是"接受"或"继续"）

节拍5：最后一个画面（可选·钩子位）
  功能：一个静止的、有情绪承载力的画面
  叙述距离：中景（拉回一点，聚焦在一个具体的东西上）
  感官主导：视觉（画面感最强的一个瞬间）
  情绪：留下余韵
```

### 核心技法
- 下降型不是"悲伤型"——下降型的目标是让读者从高处下来，不是让他们更悲伤
- 节拍3（微光）是关键——如果没有微光，下降型就会变成"压抑型"。那也可以，但要清楚你在做什么
- 环境描写在下降型中有特殊的"容器"功能——它承载情绪而不说情绪

## 混合使用

大多数章节不是纯粹的上升型或下降型。一个典型的章节结构：

```
[上升型节拍1-3：开场冲突] → [下降型节拍1-3：冲突后的喘息] → [上升型节拍4-6：新的冲突升级]
```

或者反过来：

```
[下降型节拍1-2：上一章余波的延续] → [上升型节拍1-6：本章新冲突的完整弧]
```

**选择逻辑**：
- 本章核心是什么？推动冲突（上升型）还是处理前文的后坐力（下降型）？
- 如果是连续第3章上升型 → 考虑插入一章下降型为主的章节
- 如果是连续第2章下降型 → 读者需要一个新的冲突来保持兴趣

## 与L0写作意图卡的关系

在 L0 写作意图卡中选择"节奏模型"后，按模型的节拍分解预填6个节拍。不需要填满——4个也可以。但至少知道这一章的情绪是往上还是往下。
