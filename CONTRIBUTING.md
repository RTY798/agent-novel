# Contributing to Agent Novel Skill

Agent Novel Skill 是一个 Claude Code 小说创作 skill。欢迎提交改进建议、新规则、新桥段标签或爽点梯级。

## 贡献方式

### 提交新的桥段类型标签

如果你在写作中发现了一类经常重复的桥段模式，可以提议新增标签：

1. 在对应类别下（战斗/情感/信息/过渡/弧光）新增一行
2. 提供标签名称（如 `[战斗-逃亡]`）
3. 附一个简短的使用场景说明
4. 提 PR 到 `SKILL.md` 的相应章节

### 提交新的 QC 检查项

如果你发现了一类 AI 写作中反复出现的低级错误：

1. 在 Issue 中描述这个错误模式
2. 提供 1-2 个真实例子（可以匿名化）
3. 如果是可脚本化的检查，欢迎直接提 PR 到 `quality-check.sh`

### 提交新的爽点梯级

当前的战斗 5 级 + 情感 4 级可能不覆盖所有场景：

1. 说明新梯级所属类别（如：悬疑/幽默/世界观揭示）
2. 按 L1→L4 或 L5 给出每级执行指导（1行，不含糊）
3. 提 PR 到 `SKILL.md` 的爽点梯级章节

## 提 PR 规范

- 一个 PR 只做一件事
- 修改 `SKILL.md` 时同步更新对应的 `docs/` 文件
- 如果是新规则，附一段 300 字以内的"为什么需要这条规则"说明
- README 的更新需要中英双语同步

## Issue 分类

| Label | 用途 |
|-------|------|
| `bug` | quality-check.sh 脚本问题 |
| `enhancement` | 新规则/新标签/新梯级建议 |
| `documentation` | 文档改进 |
| `writing-experience` | 使用体验分享（非 bug） |
| `discussion` | 写作方法论讨论 |

## 行为准则

- 讨论聚焦写作工艺，不评价作品内容
- 规则建议要有具体例子支撑，不空谈理论
- 欢迎不同意见，但用"我试过X效果更好"替代"你的方法不对"
