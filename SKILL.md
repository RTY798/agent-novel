# Agent Novel Skill v10.3 — 万能小说创作技能

> 从一句话想法到完整项目。任意类型。7条规则过滤AI腔。核-心写作法（角色内心运动驱动）——三种核型（变化/失败/深化）+ 防退步三机制 + 感情线核-心化。

---

## 启动模式

### Bootstrap：开始新小说项目
触发：用户说"开始新小说/我想写小说/创建小说项目"

```
第1轮（4个必答问题）：
  ① 小说标题/暂定名
  ② 类型/风格：[奇幻/都市/古风/科幻/悬疑/末世/言情/历史/武侠/轻小说/其他]
  ③ 一句话核心前提（可填"未定"）
  ④ 主角名字+一句话标签（如"林晚，末世快递员"）

→ 收集完毕 → 自动生成最小可用项目（目录结构+MASTER_SETTING+所有追踪文件）
→ 提示："说'写第1章'开始，或说'完善设定'深入规划"

第2轮（可选·仅当用户说"完善设定"/"帮我详细规划"时触发）：
  目标规模 / 叙事视角 / 世界观类型 / 主要冲突类型 / 配角 / 类型特定信息
  （奇幻：魔法体系？言情：CP数量？科幻：技术设定？悬疑：核心谜团？……）
```

Bootstrap 使用 `templates/MASTER_SETTING.template.md` 和 `templates/project-files.template.md`（按 `### FILE:` 分隔符拆分为8个文件）。

### Write：继续写章节
触发：用户说"写第X章/写下一章/续写" → 执行6步流水线。

## 7条通用硬性规则（基线）

1. **抽象情绪词0容忍**：感到/觉得/知道/认为/明白 — 禁止出现在叙述中
2. **"不是X是Y"句式 ≤ 2次/章**
3. **同一角色连续说话 ≤ 3句**（中间必须插入动作/环境/沉默）
4. **段落 ≤ 250字**
5. **同一句式不连续复用**（开头/结尾/段落结构）
6. **不写角色解释他人心理状态**
7. **核-心驱动**：不写没有"核"的场景——核=角色在这一场的内心变化（从X到Y）。找不到核→不写这个场景。**禁止用"沉默+动作细节"万能公式处理所有情感场景**（详见 `references/nucleus-first-method.md`）。NPC死亡场景必须满足三条铁律（生前回响/主角特写+错动作/性格回响）+ 一条禁令（禁止群像喊名字）。详见 `references/golden-finger-techniques.md`

> 各类型的调整建议见 `references/writing-rules.md`（建议，非强制。最终裁决权在作者。）

## 禁用词表（通用基线）
- 抽象情绪：感到、觉得、知道、认为、明白
- AI腔：仿佛、似乎、在...中
- 各类型特有禁用词见 `references/genre-guide.md`

## 6步流水线（硬性执行）

> ⚠️ **每章必须严格遵守。缺任何一步→该章不完成。**
> 以下每一步都在 Ch.47 写作清单中验证通过。不允许跳过、压缩、合并。

### 1. 前章记忆补充 + 文件加载（写前必做，不可跳过）

**⚠️ 硬性要求：读上一章全文（不是扫摘要，是读原文）。完成以下三项才能进入第2步：**

**① 找到5个身体接缝**
上一章结尾时每个角色的身体状态——手指是松开的还是攥着的、耳机是戴着还是摘下的、菜单本上是空白还是写了字——进入本章时，这些身体状态必须在。
例：Ch.46宇轩手指一根一根松开→Ch.47他握筷子之前先试了一下才夹菜。
例：Ch.46魏小丁左耳放在枕头上→Ch.47她碰左耳时耳廓是空的。
例：Ch.46嘉豪写了"到"→Ch.47他翻开菜单本会先看到那个字。
找不到5个身体接缝→退回重读上一章。

**② 找到1个氛围接缝**
上一章结尾的氛围是什么？安静/安全/私密？紧张/危险/孤立？本章结尾的氛围是延续还是打破？
例：Ch.46"今夜是末日后第一张床"（安静安全私密）→Ch.47"走廊上有八个一样长的影子"（安静安全但不再私密）
找不到氛围接缝→退回想清楚再写。

**③ 确定本章叙述焦距**
- 主视角角色是谁？（这一章情感核心属于谁，就用谁的眼睛看）
- 叙述者站在距他多远的位置？（贴肩/借眼/半空——三种焦距选一种，全章保持一致）
- 例：Ch.46半空（无人机掠过五扇门）→Ch.47贴宇轩肩膀（离他一拳距离，他看不到的读者不知道）

**然后加载文件（按优先级）**：
- **必读**：MASTER_SETTING + current_state + 前3章全文（不是摘要）+ chapter_summaries.md最近5章
- **必读**：`references/nucleus-first-method.md`（核-心法）
- **必读**：`references/nucleus-to-technique-index.md`（6技法→核型匹配表）
- **必读**：`references/seven-knives.md`（7把刀——写作困境→技法映射）
- 按需：tracking/下对应文件 + arc_maps/arc_map_卷X.md
- 裁剪规则：上下文接近15k token时，按序裁剪——③非出场角色弧光 ④检索结果。**绝不裁剪** MASTER_SETTING / current_state / 活跃伏笔 / nucleus-first-method.md / nucleus-to-technique-index.md / seven-knives.md

### 2. 核心卡片 + 技法选择（写前必填，不可跳过）

**核心卡片（本章写前必填，每项必须填，不能留空）**：
```
## 核心卡片

### 本章核
一句话：角色在本章经历了什么内心运动？
（格式：[感官载体] + [角色] + 从__到__。把感官载体嵌入核本身，让核有质感。
  错误："八个人重新感觉到自己是人"（太抽象）
  正确："八个人通过热水澡、干净床单、柠檬味洗洁精——重新感觉到自己是人"（有质感）

### 场景核（每场一句话，标明核型）
核型：🔄变化型 / ❌🔄试图变化但失败型 / ⬇️深化型
1. [核型] [角色]从__到__（或试图从__到__/或从__到更深__）
2. [核型] [角色]从__到__
3. [核型] [角色]试图从__到__但卡在__

### 感官载体
每个场景选1个承载核的具象物（最多2个用于对比/接力——不是罗列全部感官，是选最能承载核的那一个）：
1. ______（为什么是这个？因为它对角色意味着什么）
2. ______（同上，仅当两种感官的张力承载核时）

### 感情线距离
这一场结束后，两人之间的心理距离近了还是远了？______（写"近了/远了/没变"——没变说明这段感情戏没推）
```
**写前必问**：每个场景的核是三种核型中的哪一种？找不到内心运动→不写这个场景。

**核心卡片完成后——查表选技法**：
打开 `references/nucleus-to-technique-index.md`，按每个场景的核型查快速查询表：

| 核型场景 | 优先用 | 也可用 |
|---------|--------|-------|
| 压抑/克制/不敢释放 | 手法1：未完成动作 | 手法6：不可靠的身体 |
| 状态切换 | 手法2：错位的感官 | 手法5：对话的打断与留白 |
| 行为反复 | 手法3：重复中的变化 | 手法4：物的自主叙事 |
| 情感在物上 | 手法4：物的自主叙事 | 手法2：错位的感官 |
| 关系距离变化 | 手法5：对话的打断与留白 | 手法1：未完成动作 |
| 控制vs本能 | 手法6：不可靠的身体 | 手法1：未完成动作 |

**然后打开 `references/seven-knives.md`，按写作困境查刀**：

| 写作困境 | 用哪把刀 |
|---------|---------|
| 极端情绪不能直接描写 | 刀一：通感置换 |
| 角色在压抑不敢表露 | 刀二：环境反射 |
| 群像场景没有焦点 | 刀三：视角滑轨 |
| 物件没有厚度（切片式描写） | 刀四：物的时间线 |
| 过渡章节需要呼吸感 | 刀五：压缩时间 |
| 沉默已用滥 | 刀六：意外回应 |
| 卷终/大事件后需要段落感 | 刀七：全景收束 |
| 需要比喻但不想用"像……一样" | 手法：功能比喻/系统比喻/环境拟人/不可靠比喻 |

**技法组合规则**：
- 相邻技法可叠加，对角线技法可叠加，对面技法不能同时用
- 已验证的高效组合：①未完成动作+不可靠的身体 ②物的自主叙事+错位的感官 ③重复中的变化+物的自主叙事
- 禁止组合：未完成动作+对话打断（都打断都留白，节奏碎了）
- **有些场景不需要技法**：传递信息类场景（物资盘点/路线确认/信息交换）→白描。不选技法。

详见 `references/nucleus-first-method.md`、`references/nucleus-to-technique-index.md`、`references/seven-knives.md`。

### 3. 逐拍写作 + 写中信号词检测（每拍指定技法+陷阱+防滑回）

**每一拍的结构**（核心卡片中的每一个场景核↔一个写作拍）：
```
【场景名】（~字数）
技法：指定技法（来自第2步的选择）
角色在此处的身体记忆：（来自第1步的身体接缝——上一章结束时ta的身体状态是什么）
写的时候注意：（这个场景最容易掉进去的陷阱）
最容易滑回去的时刻：______（选一个最可能用老公式的位置）
替代方案：______（滑回去之后怎么拉回来）
```

**写中信号词检测**——全文扫描，每个高亮位置必须判断"有意识选择还是肌肉记忆"：

| 信号词 | 问题 | 替代方案 |
|--------|------|---------|
| 他没说话 | v10.0万能默认。他是真的不想说还是我懒得想？ | 写他身体在说什么（手/呼吸/姿势/动作）。或者用**意外回应（刀六）**——让他做一个动作，这个动作本身就是回答 |
| 她看了很久 | v10.0万能填充。她看到了什么？看的时候什么变了？ | 写她看的过程中做了什么**替代"看"**——"他把每个人夹了一筷子菜。不是'他看了每个人'，是'每只碗里都有他夹的菜'" |
| 意思就是 | 解释性收尾。作者替读者总结 | 直接删掉。让场景自己站在那 |
| 他知道/她明白 | 解释性心理。写角色内心替作者说话 | 删掉。用行为证明他知道 |
| 他坐了很久/站了很久 | 时间填充。这段时间里什么在变化？ | 写变化本身。一粒灰尘在光线里落下来的轨迹。一片叶子被风推着转了一圈 |
| 像……一样 | 常规比喻，通常多余 | 删掉"像"字，直接并置。或者用**功能比喻**——比喻的A不是装饰，是解释B的功能 |

**硬性字数**：1800-5000字。严格遵守7条通用硬性规则 + 写作技法。

### 4. 质检（4e六项 + ⑦读感检查）

- **4a**：运行 quality-check.sh（7项自动扫描，2秒）。**硬性FAIL**（抽象情绪词/不是X是Y）不达标→修改→最多3轮。第3轮仍未通过→标记[QC-FAIL]→人工裁决。
- **4a2**：韵律粗检（30秒·非硬性）。段落长度标准差+连续短句（≤8字）计数。仅统计输出，不判定PASS/FAIL。
- **4b**：命名一致性（30秒）。扫本章人名/地名，确认无意外变体。
- **4c**：桥段重复（30秒）。对照 chapter_summaries.md 最近20章[桥段类型]标签。
- **4d**：物理/数字/元叙事自洽（30秒）。数字声称 vs 实际一致；物理约束合理；无"第X章""前文提到"等元叙事用语。
- **4e**：核验 + 防退步（30秒·PASS/FAIL）。6项：
  ① **核型可辨**——遮住描写+动作，仅看行为/对话/身体反应——能看出变化/失败/深化吗？
  ② **感官载体独特**——删除它会降重吗？删了没感觉→这个感官是多余的。
  ③ **公式检测**——连续3次"他没说话/她看了很久/他坐了很久"→至少2处重写。解释性收尾→删除。
  ④ **感情线核验**——本章有感情场景吗？心理距离近了/远了？**没变→缺核，退回修改。**
  ⑤ **NPC铁律**（如适用）——生前回响/主角错动作/性格回响/禁止群像喊名字——全部满足？
  ⑥ **信号词高亮**——全文扫描信号词，每个高亮位置判断：有意识选择还是肌肉记忆？

- **⑦ 读感检查（PASS/FAIL——硬性，与4e同等待遇）**：
  把本章从头到尾**朗读一遍**（出声）。不设量化指标，但以下三项必须全部回答"是"：
  a) 读的时候有没有哪个地方需要停下来喘气？→没有说明节奏太密。
  b) 读的时候有没有哪个地方像在赶路？→有说明情绪不到位。
  c) 结尾能不能站住？→读完之后闭眼三秒，三秒内出现的画面是什么？如果画面是本章最强的那个意象→结尾站住了。如果是别的→结尾不够重，重写收尾段。

未达标→退回修改（与4a同等待遇，3轮熔断）。

详细说明见 `references/qc-rules.md`、`references/nucleus-first-method.md`、`references/seven-knives.md`。

## NPC死亡场景铁律

任何重要NPC死亡场景必须满足以下三条硬性规则+一条禁令：

- **铁律① 生前回响**：NPC生前的习惯/物品/承诺在死亡时刻回响
- **铁律② 主角特写+错动作**：至少一个主角的直接反应镜头（非群像），且必须有一个主角做错一个动作
- **铁律③ 性格回响**：死亡方式与性格缺陷或最大愿望相关
- **一条禁令**：禁止"一群人冲上去喊名字"+禁止"有人/某个/一群"等模糊人称代替具体角色名——那是噪音，不是情感。替代：单角色特写 + 错动作

> 详见 `references/golden-finger-techniques.md` 第四章。

## 分镜头章节规范

**何时使用分镜头**：
- 多线并行（如主线+支线同时发生重要事件）
- 群像独处（如Ch.46六扇门六个房间）
- 死亡序列（密集牺牲需各自专属空间）

**切换规则**：
- 用 `###` 分隔不同镜头
- 每个镜头一个核心角色（不塞两个主角的内心戏）
- 长度递减：最重的镜头放最后
- 感官锚点切换：不同镜头用不同感官做锚点

> 详见 `references/golden-finger-techniques.md` 第五章。

## 连贯性检查

**每5章**：
- 过去5章中，每2章是否至少推进一次主线？
- 是否有连续2章以上无剧情推进？

**每10章**：
- 过去10章中，每个主角是否都有高光时刻？
- 是否有主角连续10章以上被边缘化？
- 金手指手法审计：各手法使用频率是否均衡？是否有惯性/缺失？
