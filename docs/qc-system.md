# 5 层质检体系

> Agent Novel Skill 的质检不是"写完后检查错别字"。它是一个分层防御系统——自动化脚本处理可量化问题，人工目检处理需要判断力的微妙问题。

## 质检架构

```
写完整章
  │
  ├→ 4a. quality-check.sh（自动·7 项指标）—— 2 秒
  │      FAIL → 修改 → 重跑（最多 3 轮）
  │      第 3 轮 FAIL → [QC-FAIL] → 人工裁决
  │
  ├→ 4b. 命名一致性目检（人工·30 秒）
  │
  ├→ 4c. 桥段重复目检（人工·30 秒）
  │
  └→ 4d. 物理/数字/元叙事自洽（人工·30 秒）
```

## 4a. quality-check.sh — 7 项自动化检查

```bash
bash scripts/quality-check.sh "chapters/第X章-标题.md"
```

| # | 检查项 | 标准 | 类型 |
|---|--------|------|------|
| 1 | ——数量 | 20-35 个 | 风格指标（可调） |
| 2 | "不是/是"句式 | ≤ 2 次 | 硬性规则 |
| 3 | 抽象情绪词 | 0 | 硬性规则 |
| 4 | 正文字数 | 1500-5000 | 字数约束 |
| 5 | 一句话段落 | ≥ 5 个 | 节奏指标 |
| 6 | 最长段落 | ≤ 250 字 | 硬性规则 |
| 7 | 连续逗号无句号 | 0 行 | 标点规范 |

**输出示例**：

```
==========================================
质检: 第78章-长乐余烬.md
==========================================
PASS: ——数量 = 28 (20-35)
PASS: 不是/是句式 = 1 (≤2)
PASS: 抽象情绪词 = 0
PASS: 正文字数 = 3247 (1500-5000)
PASS: 一句话段 = 23 (≥5)
PASS: 最长段落 = 178字 (≤250字)
PASS: 连续4逗号无句号 = 0
==========================================
结果: ALL PASS ✓ (7/7)
```

### 熔断机制

最多 3 轮 QC 修改。第 3 轮仍未通过 → 标记 `[QC-FAIL]` → 交作者人工裁决，不阻塞流水线。

**为什么是 3 轮**：无限循环修改是 AI 写作的陷阱——有时候一段文字就是通不过某个检查，但硬改成"合规"会让文字更差。3 轮上限保护文字质量。

## 4b. 命名一致性目检（30 秒）

**检查什么**：扫本章主要人名/地名，确认无意外变体。

**真实案例**：
- "秦队"在某处写成了"秦队长"且非有意为之 → 修复
- "祺"在某处误写为"祁" → 修复
- 角色别名在不同场景一致性（"安凯尔"/"凯尔"）

**不需要检查的**：有意为之的变体（正式场合叫全名 vs 私下叫昵称）。

## 4c. 桥段重复目检（30 秒）

**检查什么**：对照 chapter_summaries.md 最近 20 章的 [桥段类型] 标签，确认本章无显性重复。

**17-18 个标签分类**：

| 类别 | 标签 |
|------|------|
| 战斗 | [战斗-被碾压] [战斗-险胜] [战斗-智取] [战斗-碾压] [战斗-不战而胜] |
| 情感 | [情感-被理解] [情感-和解] [情感-回应] [情感-清算] |
| 信息 | [信息-揭露] [信息-伏笔] [信息-回收] |
| 过渡 | [过渡-移动] [过渡-休整] [过渡-搜索] |
| 弧光 | [弧光-打破] [弧光-重建] [弧光-推进] |

**为什么是 20 章**：人的阅读记忆大致覆盖最近 20 章的印象。超过 20 章前的桥段重复，读者不太会察觉。

## 4d. 物理/数字/元叙事自洽（30 秒）

### 数字自洽
叙述中声称的数量与实际是否一致。
- ❌ "四个字"实际是五个字
- ❌ "三个人"实际数出四个人
- ✅ 声称数量 = 实际数量

### 物理约束
载具容量/空间容量/时间距离是否合理。
- ❌ 5 座比亚迪坐 8 个人
- ❌ 半天走 500 公里
- ✅ 所有物理描述符合 current_state.md 登记的载具/物资/时间

### 元叙事污染
正文/对话中是否出现了不应出现的叙事层用语。
- ❌ "这是第 12 章给你的"
- ❌ "前文提到"
- ❌ "如前所述"
- ❌ 角色说"这一章的标题应该叫XX"

**这是 78 章连载中踩过最隐蔽的坑**——角色不能知道自己在小说里。

## 为什么不是 6 层或 4 层

5 层是一个平衡点：
- 少于 4 层 → 漏掉可预防的错误
- 多于 6 层 → 每章质检时间超过 5 分钟，写作节奏被打断

当前 4a（2 秒自动）+ 4b+4c+4d（90 秒人工）= **约 2 分钟/章**。这是可接受的"打烊前检查"时间。
