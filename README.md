# Agent Novel Skill v10.3

> 万能小说创作技能 —— 从一句话想法到完整项目，AI 写作的反套路引擎。

## 一句话介绍

不是帮你"写小说"，是让 AI 写出**没有 AI 味**的小说。7 条硬性规则 + 核-心写作法 + 6 步流水线，专为番茄小说等免费阅读平台优化——每章都要让读者点"下一章"。

## 为什么需要这个 Skill？

AI 写小说最大的问题是"一眼就能看出是 AI 写的"：

- "仿佛""似乎""或许"满天飞
- 角色只会"沉默+动作细节"万能公式
- 章节有开头有结尾但中间没有"核"——角色从头到尾没变过
- 战斗场景像在"报告发生了什么"而不是"经历一场战斗"

这个 Skill 就是解决这些问题的。

## 核心特性

### 7 条硬性质检规则

| # | 规则 | 阈值 |
|---|------|------|
| ① | 破折号 `——` | 20-35 次/章 |
| ② | "不是X是Y"句式 | ≤2 次/章 |
| ③ | 抽象情绪词（感到/觉得/知道/认为/明白） | 0 次（叙述中） |
| ④ | 正文字数 | 1500-5000 字 |
| ⑤ | 单行段落 | ≥5 个 |
| ⑥ | 最长段落 | ≤250 字 |
| ⑦ | 连续逗号不换句 | 禁止连续 4 个逗号 |

附带 `quality-check.sh` 脚本，一键检测全部 7 条规则，逐条 PASS/FAIL。

### 核-心写作法（Nucleus-First Method）

**不写没有"核"的场景。** 核 = 角色在这一场的内心变化（从 X 到 Y）。

三种核型：
- **变化**：角色在这一场改变了什么
- **失败**：角色想改变但没改成——是没改成才推动了剧情
- **深化**：角色选择不改变——用新的细节强化已有特质

防退步三机制 + 感情线核-心化，拒绝模式化写作。

### 6 步流水线

```
前章记忆补充 → 核型设计 → 正文写作 → 质检 → 文件更新 → 核验
```

每步硬性执行，不允许跳过或合并。已在 60+ 章真实小说写作中验证。

### 七大金手指手法体系

7 把刀 + 5 种比喻手法 + 感官变形 + 分镜头规范 + NPC 死亡铁律

### Bootstrap 系统

4 个问题启动新小说项目，自动生成完整目录结构、MASTER_SETTING、8 个追踪文件。

## 适用场景

- 番茄小说 / 起点 / 晋江等网文平台写作
- 免费阅读平台的高完读率写作
- 长篇连载（50-100 章）项目
- 所有类型：都市 / 奇幻 / 末世 / 古风 / 科幻 / 悬疑 / 言情 / 轻小说

## 实战验证

已在真实小说项目中验证 60+ 章：
- 每章质检 7/7 PASS
- AI 腔禁用词 0 违规
- 战斗场景六维覆盖
- 角色性格锚点逐章追踪
